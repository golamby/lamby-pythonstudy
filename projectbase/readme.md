# 项目需求

    1.额度15000或自定义  
    2.实现购物商城，买东西加入购物车，调用信用卡接口结账
    3.可以提现，手续费5%
    4.支持多账户功能
    5.支持账户间转账
    6.记录日常消费
    7.提供还款接口
    8.ATM记录操作日志
    9.提供管理接口，包括添加账户，用户额度，冻结账户等功能
    10.用户认证装饰器
